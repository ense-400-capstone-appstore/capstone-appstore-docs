GET /api/v1/groups/{id}

Get a Group by ID.

Parameters:

N/A

Authorization:

admin, owner of group, members of group

Example Request:

GET /api/v1/groups/1

Example Response:

200 OK

{
    "data": {
        "id": 1,
        "name": "Administrator's Group",
        "created_at": {
